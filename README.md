A premium, high-fidelity landing page for an advanced AI and data analytics platform. This project showcases a futuristic, immersive user interface with complex animations, 3D mockups, and high-density information layouts.

## ✨ Key Features

- **Immersive 3D UI & Mockups**: Interactive dashboard mockups with floating widgets, real-time data visualizations, and activity logs.
- **Advanced Animations**: Powered by Framer Motion (`motion/react`), featuring smooth page transitions, SVG path animations (e.g., real-time moving dots on bezier curves), and dynamic background glows.
- **Bento Box Layout**: High-density information display for platform features, including animated node networks and spinning radar effects.
- **Interactive Solutions Showcase**: Detailed feature cards with hover effects, performance statistics, and a premium glassmorphism design.
- **Smooth Navigation**: Scroll-spy navigation with dynamic gradient highlighting and smooth scrolling to specific sections.
- **Responsive Design**: Fully responsive layout optimized for all screen sizes using Tailwind CSS.

## 🛠️ Tech Stack

- **Framework**: [React 18+](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Start the development server:
   ```bash
   npm run dev
   ```

2. Build for production:
   ```bash
   npm run build
   ```

## 🎨 Design Philosophy

The design language focuses on a "Cyber-Premium" aesthetic, utilizing:
- Deep, dark backgrounds (`#05050A`) to make colors pop.
- Vibrant, high-contrast gradients (Cyan `#00F0FF`, Purple `#8A2BE2`, Pink `#FF007F`).
- Subtle glassmorphism (`backdrop-blur`) and glowing borders to create depth.
- Data-driven visual elements (charts, metrics, node networks) to reinforce the platform's technical capabilities.
